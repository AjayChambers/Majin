# MAJIN

Majin is a set of JSON tools written in C++. The core purpose of majin is to provide an interface on which
graphical software can be written by providing low level binary interfaces, and command-line level
interfaces. The tools Majin provides are intended to be fast, and robust. Though they are written in C++
they are not indended to be used as C++ libraries, but rather, as a stand-alone CLI, an ABI for C#, or as
a library for Node.js.

## About
Originally the project started out as JSON interpreter software. As the project was developed, a JSON AST
generator was developed for it, and later JSON manipulation features/tools were written into the Jin CLI.
With M from 'M'anipulation, and an A from 'A'ST generator, the name Majin was born.
